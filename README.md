# Assets

This folder holds all static assets that are not HTML pages.

## Structure
- `images/vacuum-mug/`: Product imagery for the Vacuum Mug (Thermobecher). The files currently ship as SVG placeholders so the layout is stable. Replace them with the real JPEG/PNG versions when available, keeping the same filenames:
  - `mug-hot-cold.svg`
  - `mug-wide-mouth.svg`
  - `mug-feature-icons.svg`
  - `mug-camp.svg`
  - `mug-car.svg`
  - `mug-product.svg`

Feel free to add higher-resolution versions in the same folder; the HTML uses relative paths from `assets/images/vacuum-mug/`.
