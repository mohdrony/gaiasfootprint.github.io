# Assets

This folder holds all static assets that are not HTML pages.

## Structure
- `images/vacuum-mug/`: Product imagery for the Vacuum Mug (Thermobecher). The files are provided as JPGs:
  - `mug-hot-cold.jpg`
  - `mug-wide-mouth.jpg`
  - `mug-feature-icons.jpg`
  - `mug-camp.jpg`
  - `mug-car.jpg`
  - `mug-product.jpg`

Feel free to add higher-resolution versions in the same folder; the HTML uses relative paths from `assets/images/vacuum-mug/`.
